# 项目上下文信息

- 用户正在创建HarmonyOS火车票查询系统的渐进式教程网站。采用Tab标签式布局，每个Step显示该步骤的完整代码（不是片段）。
1. 显示出发地、目的地功能实现（10 分）
出发地和目的地数据通过读取 entry/src/main/resources/rawfile 中 city.json 文件获取，文件读取
成功后，使用 JSON 解析成实体对象，从实体对象中获取城市数组，单击出发地，弹框显示城市列
表，选择城市后，将选择的城市显示在页面中；单击目的地，弹框显示城市列表，选择城市后，将
选择的城市显示在页面中。将页面显示、城市弹框列表选择、读取 city.json 的实现代码截图并提交
到答题框；
